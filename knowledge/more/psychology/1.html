# 精神分析引论

# 误差心理学

# 第一讲

我们首先需要克服对精神分析的本能反对与抗拒。

身与心之间的关系，身心失衡是精神异常的关键，精神分析的目标就是填补这个差距。

精神分析认为人的性冲动对人类心灵文化，艺术和社会做出了巨大的贡献。

每人个希望自己获得文明的成果，竭力反对并害怕暴露他们的性本能。

总结：自我反对 -> 没有客观依据 -> 自我分析 -> 社会反对 -> 困难 -> 性冲动 -> 偏见

# 第二讲

  • 误差性感觉

什么是误差性感觉？

- 说错话、写错字、放错东西、听错意思、叫错名字。

跑偏本义就是有意，误差不是意外，而是有效的心里行为，有内在含意。

精神分析观察生活中很琐碎的事情，细小的迹象。

# 失忆错误

  • 想不起来对方的名字，是因为没有好感，不愿意回忆这些内容。

  • 计划遗忘，是因为一种对抗情绪导致不希望执行计划。

丢失物品

- 丢失他人赠送的物品，与有意甩掉有关联，不再希望留作纪念。
- 物主不再喜欢自己的旧物，希望找一个借口获得较新较好的物品。或者让物品损坏也可以达到同样的效果。
- 随着动机消失，无法找到的、错放了的物品也会出现。
- 反复忘记和组合忘记，健忘是一种选择。

# 第四讲

误差是有意的，是一种心理行为

错误是两个不同意图相互干扰的结果，其中一个是有意干扰，另一个为干扰意图。

# 什么样的意图，使干扰源产生？

抑制现在说话的意图，是口误产生的必要条件。

# 干扰与被干扰是怎样的配比关系？

对立、更正、补充

几乎所有的口误，都是人所说的、与他想要说的，正好相反。误差是一种冲突的表现。

忘记名字，是因为他本身就不喜欢他们，或因为他们让他想起一些不喜欢的事情。
