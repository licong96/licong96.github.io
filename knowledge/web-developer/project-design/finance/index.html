# 财务记录管理系统

# 想法与需要

  • 我做什么？

    1. 用可视化图表展示我每个月的支出，和收入。
    2. 我可以清晰的看到每一笔支出的细节。
    3. 我可以对数据进行多种比较。
    4. 我可以看到数据的消费趋势。
    5. 观察自己的消费习惯。

  • 网站如何实现我的目标？

    将我手动写入表格中的数据，上传存储到数据库，再从数据库中拿出数据，给前端用图表可视化的方式展示出来。

  • 做什么，以怎样的顺序，才能达成我的目标？

    1. 准备一个精细的表格。
    2. 将表格上传，解析成有效的数据。
    3. 将数据存储到数据库中。
    4. 设计可视化图表和数据结构。
    5. 用前端页面将数据展示处理。
