# Node

  • NodeJs是什么？

    它是一个javascript的运行环境

  • 它的主要用途有哪些？

    1. 运行在本地，作为构建打包工具
    2. 运行在服务器，作为web server

# NodeJs与JavaScript的区别

  • 相同点

    • 都使用了ECMAScrip定义的语法规范。

  • 不同点

    • JavaScript = ECMAScrip + Web API
    • NodeJs = ECMAScrip + NodeJs API
