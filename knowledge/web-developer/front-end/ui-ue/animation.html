  • 支持 CSS，DOM，SVG，和JS对象

  • 运动图形库

  • 与Anime.js类似

  • 用TypeScript编写

# Pixi.jsopen in new window

  • 快速的轻量级2D动画渲染引擎。
  • star 33.6k

# matter.jsopen in new window

  • 接近现实生活中的物理运动、碰撞、惯性动画库

# snabbt.js

  • 简单方便的写复杂的动效组合
  • 大小5K

# AniJS

  • 通过JavaScript控制的动效库
