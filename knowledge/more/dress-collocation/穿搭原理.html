# 视觉原理

# 蓬佐错觉

也叫对比错觉，在两根透视线的影响下，下面的会显得更短。

  • 解决髋大腿粗问题

    • A字型的裙摆或阔腿裤
    • 上窄下宽，视觉显瘦

  • 上宽下窄穿搭

  • 上紧下宽穿搭

# 菲克错觉

也叫水平垂直视错觉，人看垂直线比看水平线更加费力，会觉得垂直线更长一些。

  • 让身材显得更高更修长

    • 穿外套或大衣时把衣服敞开

      • 不系扣子

    • 内搭同色或相近色更显高

      • 因为同色系没有被切割，视觉垂直线更长
      • 留出腰线

    • 鞋裤同色

      • 也是让垂直线更长

  • 竖线条给人感觉更顺畅，更显高

    • 上下衣勾勒出大型体上的竖线条

      • 一两条就够，避免有很多竖线

# 缪勒莱尔错觉

  • 上V下A的穿搭更显高

    • 衬衫，解开上面和下面的扣子
    • V领马甲
    • 下摆微喇的阔腿裤

# 艾宾浩斯错觉

也叫对比错觉，周围参照物的不同，会使人对大小的认知产生偏差

  • 宽松+紧身的对比

    • 宽松的服装往往可以显得更瘦

  • 用小格子花纹来衬托自己，更有立体感

  • 用更大的穿搭来变小

    • 腿粗就穿宽松的阔腿裤
    • 腰粗就穿下摆大的裙子
    • 手臂粗就穿宽阔的袖子

# 亥姆霍兹错觉

密集的横条纹，有横向挤压，纵向拉伸的视觉效果。

  • 横条纹的衣服更能显出身材的高挑

    • 粗细和密集程度

      • 窄且密集的横条纹显瘦、显高
      • 宽且稀疏的横条纹显胖、显矮

  • 竖条纹，相反

    • 越宽、越稀疏的竖条纹反而更加显瘦、显高
    • 越窄，越密集的竖条纹反而显胖、显矮

# 凯尼泽三角错觉

  • 只要能显瘦，就能同时显高

    • 头小人就显高

  • 遮住最胖的部位，露出最细的部分就能显瘦

    • 露出脖颈
    • 露出手腕
    • 露出脚踝

# 视觉焦点原理

  • 引导眼睛的重心往上看，可以显高

    • 鲜艳的颜色在上身

    • 配饰

      • 项链
      • 帽子
      • 眼镜
      • 丝巾丝带
      • 别针
      • 围巾

    • 在上身搭一件衣服

  • 避免视觉重心往下

    • 不要穿花哨的裤子
    • 不要穿吸引眼球的鞋子
