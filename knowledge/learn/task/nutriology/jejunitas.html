# 断食

一日三餐的生活方式只有几百年的历史，而一日两餐或一餐是持续了几万年的历史，在我们的基因里早就适应了，反而是一日三顿，我们还没有完全适应。

# 好处

  • 断食可以让细胞进入修复状态，可以治疗很多疾病
  • 少吃能长寿、保持年轻
  • 体重降低、血压降低、血糖降低
  • 酮体增强，消耗脂肪来提供能量

# 研究人员

  • Otto Buchinger
    • FrançoiseWilhelmi de Toledo
  • Valter Longo教授

# 断食研究

  • 每天可以喝2-3L的水
  • 摄入200-250kcal的食物
  • 碳水化合物摄入量每天为25-35g

长时间断食（2-4天）可以使整个免疫系统再生，并且逆转化学疗法对免疫系统的损害。

隔天断食法能促进产生棕色脂肪。

在6-24周不等的时间里，间歇性断食可以帮助减少内脏脂肪4-7%。

断食的时候激发细胞自噬，能启动细胞进行自我修复，还能刺激生长激素，这会让身体长出更多新细胞，让我们更显年轻。

# 重点

  1. 缩短进餐窗口期。
  2. 断食期间卡路里控制在500cal以内
  3. 超过24小时的断食是可以，两天有点长
  4. 多吃营养密度高的食物，保证营养的前提下，断食或者少吃。
  5. 注意吃水果，因为糖分太高，影响效果

# 早餐

如果上午能维持良好的工作状态、没有出现低血糖的状况，是可以不吃早饭的话。比早餐更重要的是一天整体的饮食质量。

# 16:8

  • 一日两餐。
  • 进餐窗口期：12-20点，8小时进餐，16小时断食。
  • 吃午饭与晚饭，适合有运动量的一天。

# 20:4

  • 一日两餐。
  • 进餐窗口期：8-12点，4小时进餐，20小时断食。
  • 吃早饭与午饭，适合没有运动量的一天。

# 23:1

  • 一日一餐。
  • 第一天下午6点左右吃饭晚，1小时进餐，23小时断食。
  • 第二天，禁食早饭，午饭，到下午6点左右吃晚饭，补充一天营养。

注意：

  1. 不要剧烈运动，以免低血糖和肌肉分解过多。
  2. 断食后的第一餐很关键，营养要搭配好，不要暴食，不要吃甜食。
