# 字段说明

  • 场地ID字段 - _field
  • 场地时间字段 - _time

# 规则

提前7天预订，预订时间为7:00

锁住课程，拿一天的某时间做实验，找一个比较容易约出去的一个时间段，用程序锁住，一直在等待支付中。

# 捡漏

循环调用接口，针对周六下午，14-15 or 15-16

获取周六的列表接口，遍历10个场地，两个时间段，判断是否能够预约，如果可以就直接下单，如果不能，重新获取最新列表数据，继续判断。

  • 列表接口6分钟调用一次，一天调用240次
  • 列表接口3分钟调用一次，一天调用480次
