# access_token

和微信交互的钥匙，几乎每个接口都会用到

  • 每2个小时失效，需要重新获取
  • 更新之后，之前的就不能再用

我们的系统每个2小时启动去刷新一次access_token，这样可以保持最新的。为了方便频繁调用，需要把access_token存储在一个文件里。
