# 大前端

现在主流的项目结构分为三层：

  1. 前端
  2. 后端
  3. 服务端

大前端开发者需要做前两件层，用 Node 做后端，写接口，拿数据，相当于一个中间层。前面两层关注的都是业务层。

真正的后端开发者只需要关注服务端。
