# 食物

  • 食物多样性，每天收入12种以上的食物，每周25种以上。

# 人体所需的营养素

三大宏观营养素：

- 蛋白质、脂肪、碳水化合物（糖）

维生素：

- A、B、C、D、E

矿物质：

- 常量元素：钙、磷、钾、硫、氯、镁 
- 微量元素：铁、铜、锰、锌、 钴、钼、铬、镍、钒、氟、硒、碘、硅、锡

# 七大营养素

水

- 水的摄入量不足时，血粘度增加，长期会引发高血压，还会影响排泄，尿少废物积存，结石，代谢中毒，而引发多种疾病
- 水摄入过多可导致水中毒

蛋白质

- 蛋白质摄入过多易致便秘、食欲缺乏
- 过少易致抵抗力下降、肌张力下降、水肿、贫血、消瘦

脂类

- 摄入脂肪过多易引起食欲下降、肥胖等疾病。
- 摄入不足容易导致营养不良，导致皮肤粗糙等皮肤疾病

碳水化合物

- 碳水化合物的主要功能是为人体提供能量，是三大营养素中最廉价的营养素，是人体获得能量的主要来源。
- 供给过多，易致体重增长过快、虚胖、苍白、肌张力下降。
- 供给不足，减少脑细胞，肌肉疲乏无力，低血糖。

矿物质

- 它们本身并不供能，主要在构成人体的物质和调节体内生理、生化功能方面发挥着重要作用。

膳食纤维

维生素

  • 维生素是维持人体正常生命活动所必需的营养素，有14种。

# 五大类食物

  1. 谷薯类
  2. 蔬菜水果类
  3. 畜、禽、鱼、蛋、奶类
  4. 大豆坚果类
  5. 油脂类

# 谷薯类

补充：碳水化合物、膳食纤维、B族维生素、矿物质

  • 主食，总要的能量来源
  • 每天250 - 400克，占膳食总量的一半
  • 平均每天3种以上，每周5种以上

食物来源：

- 薯类：红薯、马铃薯、芋头、山药
- 全谷物：稻米、米饭、面条、馒头、包子、饼、饺子、面包
- 杂粮：燕麦、小米、玉米、薏米、荞麦
- 杂豆：红豆、绿豆、豌豆、芸豆
- 其它：八宝粥、二米饭、豆饭。

# 蔬菜水果类

补充：维生素、膳食纤维、糖、矿物质

  • 平均每天5种以上，每周10种以上。

# 畜、禽、鱼、蛋类

补充：优质蛋白质、维生素A/B、脂肪、矿物质

  • 平均每天2种以上，每周5种以上
  • 优选鱼和禽类，鱼和禽类脂肪含量相对较低
  • 鱼类含有较多的不饱和脂肪酸
  • 兽肉应选择瘦肉，脂肪含量较低

平均每天收入量：

- 鱼、禽、蛋、瘦肉总量：120 - 200克。

每周摄入量：

- 水产类：280 - 525克
- 畜禽肉：280 - 525克
- 蛋类：280 - 350克

食物选择

- 畜禽肉：瘦猪肉、鸡、鸭、牛、羊
- 水产类：鱼、虾、蟹、贝壳
- 蛋类：鸡蛋

# 奶、大豆、坚果类

补充：钙、优质蛋白质、维生素E、脂肪、矿物质
