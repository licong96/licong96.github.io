# 认知天性

你的智力在很大程度上由你自己控制，学习会改变你的大脑与认知。每当你学到新东西时，大脑就会发生改变。要想学以致用，就必须形成长期记忆。

耗费心血的学习才是深层次的，不花力气的学习就像在沙子上写字。如果感到学习非常吃力，那是说明你正在学习非常重要的东西。想在已有的水平上有所进步，达到真正的专业程度，就要明白努力与挫折是必不可少的。

# 效率低的方法

  • 反复阅读
  • 集中训练，机械的、快速的重复一些东西
  • 填鸭式学习

频繁的集中练习只会产生短期记忆

# 更好的方法

  • 检索式练习
  • 穿插练习有助于长期记忆
  • 多样化练习促进知识的活学活用
  • 间隔练习更牢固
  • 自测
  • 反思
  • 细化
  • 学习环境、养一个场

  1. 当遇到某个概念时，用自己的语言将它描述清楚。

  2. 把新知识转化为一系列问题，并在之后的学习中试着解答这些问题。

  3. 把新知识和已知联系起来，回顾自己以前的经历和相关经验，思考怎么运用，将知识化为自己的能力。

# 检索式练习

强化神经回路的具体方法就是：检索记忆，练习所学的东西，回想事实、概念、事件。定期练习可以防止遗忘，强化检索路径，并阻止遗忘。

我们在阅读过一段文字或听过一堂课后，只需用一道简单的小问题考考自己，就可以巩固所学、强化记忆。周期性的做一些该知识的检索练习是非常有必要的，将内容拆分，渐进式自测。

推迟的后续检索练习要比立即练习更能强化记忆，因为延迟后再检索需要花更大力气。

最为有效的检索练习，是那些可以反映出你今后如何运用自己知识的检索练习。

有时间间隔的练习会改善记忆。检索时付出的认知努力越大，记忆效果就越好。知识或技能越容易被检索，就越不容易被记住。相反，你在检索知识或技能上花费的努力越多，检索练习就越能深化这种记忆。

# 自测

把测验当成衡量一个人学习成果的方法，这让人感到害怕，反而让人放弃了这种最有用的学习工具。

自行测验要比重复阅读花更多功夫，这比较难，但在检索上下的功夫越多，记住的东西也就越多。

可以用来判断自己知道什么和不知道什么，避免被精通的假象所迷惑。

# 穿插练习与多样化练习

合理的间隔时间，穿插学习不同的东西，更有助于记忆。

锻炼识别能力和归纳能力。有助于熟练地从不同类型的问题中提炼出基本原理。

# 反思

花点时间复习最近学过的课程或取得的经验，然后自问：

  • 在哪些地方做得好？
  • 哪些事情可以做得更好？
  • 想做的更好，还需要学什么？
  • 想起了其他什么知识或经验？
  • 为了下次做得更好，要采用哪些策略？

拿一些问题考考自己：

  • 课程的核心思想是什么？
  • 哪些是相关的例子？
  • 如何把这些内容和我的已知联系起来？

人们可以通过反思自己的经历获得一种必不可少的知识。反思会涉及多种认知活动，这些活动可以带来更好的学习效果。

从记忆中检索知识或早期训练过的内容，把这些东西和最新的体验联系起来，借助观察和思考，获得新的知识，预先演练你下次可能采取的不同做法。

# 细化知识

细化就是理解新知识的过程。只要能做到细化知识，就能无止境的学下去。

细化的方法：用自己的语言把新知识表达出来，把它和已有的知识联系起来。联系的越紧密对新知识的掌握就越牢固，关联的越多越有助于记忆。

把新知识放到更广泛的情景中有助于学习。从新知识中提取关键概念，并把这些概念组织成一个心智模型，同时把这个模型和已知的联系起来。

# 错误的观念

  • 花足够多的时间重复背诵，重复的次数越多，记住的概率越大
  • 学起来速度更快，更轻松
  • 把所有注意力只用来学一项技能

# 科学的观念

  • 反复阅读课本对长期记忆没有多大帮助
  • 感觉学习更吃力，记忆才更为长久
  • 当事关重大时，当抽象的事物被形象化时，当与个人息息相关时，会把学到的东西记得更牢
  • 在初次阅读过后，隔一段时间再阅读是有意义的
  • 有计划的进行间隔练习比全部集中练习要好

# 反复阅读的不足

  • 浪费时间
  • 无法产生持久的记忆，还会带来错觉，对文本的熟悉导致自己误认为已经掌握
  • 过程看起来刻苦，但学习时间的长短不能用来衡量掌握程度

重复阅读与集中练习会让 人越做越熟练，以为自己已经掌握了知识，但实际上，这种方法达不到真正的精通，也不会产生持久的记忆，只是在浪费时间。

连续重复阅读不是有效的学习方法。多次阅读相同的课文，人们的理解以及延伸出来的看法都是一样的。而另一些研究则表明，重复阅读只有微不足道的好处。

比如自己，有尝试过花10个小时看JS文档，最后收获的到东西极少，无法内化，这种学习方法的效率极其低下。

# 学习假象

重复阅读基本没有效果，那么为什么学生们还愿意用这种方法呢？

  • 一个原因可能是他们一直在接受糟糕的学习建议
  • 还有一个原因是自己认知上的错觉

对一段文字越熟悉，越能流畅阅读，就会造成一 种假象，认为自己已经掌握了阅读的内容。能背诵课文或笔记并不代表你理解了它们所描述的要 义，也不代表你会应用这些内容，更不代表你知道如何把它们和已知联系起来。

如果不给自己出题，容易过高的估计自己对学习资料的掌握程度。很轻松的接受其中的内容，觉得自己已经很明白，大脑会放弃继续学习。

# 把考试当做记忆练习

主动检索（考试），可以强化记忆。检索花费的心思越多，受益就越多。

  • 主动检索的好处：
    • 什么是你知道的，什么是你不知道的。
    • 回想以前学过的东西，让大脑巩固记忆。

# 渐进学习，延时反馈

在查看正确答案之前，自己先进行猜测和分析好过于直接看答案。

# 间隔训练比集中训练更有效

因为向长期记忆中存放新知识需要有一个巩固的过程。在这个过程中，记忆痕迹(大脑中有关新知识的心理表征)得到加深，被赋予含义， 并和已知联系起来——这个过程需要数小时，甚至数天。

集中练习往往导致填鸭式学习，来的快，去的也快。

有间隔的安排练习，穿插其他的学习内容，加上多样化的练习，就是为了造成一段时间的遗忘，然后再付出努力回忆学过的概念，这样做获得的领悟更深，掌握得更牢固。

快速频繁的练习会产生短期记忆，持久记忆则需要花时间进行心理演练以及其他巩固过程，因此有间隔的练习更为有效。出现了一些遗忘后，再检索所学的东西，就需要花费更多的力气，这会重新触发巩固过程，进一步强化记忆。

# 穿插练习有助于长期记忆

在练习中插入两个以上的主题或技能，也是一种胜过集中练习的学习方法。

从感觉上说，用穿插安排内容的方式学习，效果要比集中练习来得慢。

多样化练习促进知识的活学活用

# 间隔多长时间才够?

只要练习不是无意义的重复就可以。从最低限度上说，间隔的时间足够出现一点儿遗忘就对了。练习环节中间出现一点儿遗忘是好事，只要它能让人更加努力地练习就行。

睡眠在巩固记忆的工作中扮演了重要角色，所以在两次练习至少间隔一天。

艾宾浩斯遗忘周期：

5分钟、30分钟、12小时、1天、2天、4天、7天、15天、30天、3个月。

根据当下情况找一个周期进行自测即可。

# 知识获取的过程

  1. 编码
  • 最开始是对短期工作记忆中信息的编码，这时信息还没有被巩固成长期记忆中的知识。
  1. 巩固
  • 巩固会辨识并稳定记忆线索，赋予其含义，把它们与过去的经验以及长期记忆中已经存储下来的其他知识联系起来。
  1. 检索
  • 检索会更新所学的东西，并让你做到学以致用。

学习、记忆、遗忘。

学习总是建立在已知基础之上。我们是通过与已知建立联系 这种方式来解读事件和记忆事件的。

只要有亲身体会，再加上时不时的练习，重要的知识、技能与经验就不会被遗忘。

# 学习中必须要做哪些“努力”

  • 生成

  • 重新巩固记忆

  • 打造心智模型

  • 举一反三

  • 构建概念学习

  • 学习迁移

  • 做好学习的心理准备

# 生成

尝试解答一道题目或是解决一个问题，而不是坐等信息或解决方案出现。

想出一个答案要比从多个选项中选择一个答案更有利于学习。进行更高层次的思考任务，好过于被动地接受他人提供的知识。

# 重新巩固记忆

检索练习。

# 打造心智模型

使彼此相关的复杂理论或是连续的运动技能融合为一个有意义的整体，这就是心智模型。

# 举一反三

在不同时机、不同环境下多次进行检索练习，其间穿插不同的学习资料，这样做有助于给这些资料建立新的联系。

# 构建概念学习

靠有间隔地、穿插式地接触样例来下定义。强化辨析和归纳这两种技能。

# 学习迁移

所谓学习迁移， 是指在新环境下运用所学的一种能力。

使用有间隔、有穿插、多样化的方式进行检索练习会带来困难。克服这种困难需要一套心理活动。

# 做好学习的心理准备

若是在没有答案的情况下自行寻找解决方案，你就能更好地理解答案，也能把它记得更牢。

在测验中，想出一个答案要比从多个选项中选择一个答案更有利于学习。解决一个问题好过于记住一个问题。尝试思考一个答案，但是得出的结果是错误的，也好过于不去尝试。

反思，回想一下自己学到了什么，拿几个问题考考自己。

不要把错误看成失败，而是将其视为通往精通之路的必修课与转折点。并且把失败看作努力的标志，失败说明有必要加倍努力，或是解放思想、尝试不同办法。

# 打造适合自己的心智模型

把知识整合起来，解决各种问题。

# 达克效应

无法胜任某项工作的人缺乏提高自己的能力，因为他们分不清能与不能之间的区别。不能胜任某项工作的人会过高地估计自己的能力，而且感觉不到自己的表现与实际要求之间的差距，觉得没有必要试着改进。

需要让人的元认知更为准确。

# 元认知

也就是我们对自己知道的，知道些什么，以及如何评估自己的表现。

只依靠我们的感知系统，即直觉、记忆与认知，这些系统都存在偏差、失误、偏见与瑕疵。我们容易受到错觉与误判的影响，因此做事时要缓一缓。

避开错觉和误判的办法是，用一组自身之外的客观标准，来替代用作决策参考的主观经验。

最重要的是经常使用测验与检索，来验证什么是你真正知道的，以及什么是你以为自己知道的。不要相信集中练习产生的临时收效。加长测验间隔，运用多样化练习，把眼光放长远一些。

# 怎么判断自己是否掌握

若是延迟一段时间后还能轻松检索，这倒是能代表你学得不错。

更好的做法是打造一个心智模型，要知道自己对一个知识理解到了什么程度，要看自己能在多大程度上对其进行合理的解释。因为解释的时候，需要回忆要点，把它们组织成自己的语言，并解释为何这才是要点。

# 认识自己的智力

如何弄明白自己需要什么知识，以及如何获取这些知识。 我擅长什么，我最在意什么，我努力的方向是什么。

# 加德纳的8种智力

  • 逻辑——数学智力: 批判性思考，以及使用数字和抽象概念的能力，诸如此类。

  • 空间智力: 三维判断，以及在脑海中具象化的能力。

  • 语言智力: 使用文字和语言的能力。

  • 肢体动觉智力: 行动敏捷和控制身体的能力。

  • 音乐智力: 对声音、旋律、音色，以及音乐的敏感性。

  • 人际交往智力: “读懂”他人，以及与人有效协作的能力。

  • 内省智力: 理解自身，准确判断自身知识、能力、效率的能力。

  • 自然观察智力: 区分和关联周围自然环境的能力

# 罗伯特的3中智力

分析型、创新型与实践型智力。由生活环境所需决定。

# 动态测试学习法

  1. 先进行测验。明白自己哪些方面的知识还有所欠缺。

  2. 然后运用检索、反思、自测、间隔、多样化等方式去练习并提高自己的知识。

  3. 再次进行测验。观察自己哪些方面有所改善，哪些地方还需努力。

# 专注于自律

勇气、好奇心与坚持。

努力学习会改变你的大脑，创建新的连接，建立心智模型，扩展你的能力。我们的智力水 平并非天生已定，而在很大程度上由我们自己发展。我们之所以努力，是因为努力本身能拓展我们的能力。 你所做的事情决定了你会成为什么样的人，决定了你有能力做什么。你做的事情越多，你能做的事情也就越多。

想要精通某事或达到专业水平， 完全不需要拥有超人的基因，但必须拥有自律、勇气，以及持之以恒的精神。通过持续、有目的的练习和反复的应用，形成更深层的编码与潜意识中的精通。

# 搭建记忆宫殿

学习助记方法。

先选择一个地点作为自己的记忆宫殿，把每一个核心观点和宫殿中的一个位置对应起来，在脑海中完成可视化，

# 学习策略建议

  1. 练习从记忆中检索新知识

  2. 有间隔地安排检索练习

  3. 学习时穿插安排不同类型

  4. 细化知识连接已知

# 1.练习从记忆中检索新知识

把从记忆中检索知识和技能当成主要学习方法。

在学的过程中时常停下来，问自己一些问题：核心概念是什么？如何定义它们？哪些是我没有接触到的？这些和我已知的有哪些联系？

每周自测学过的东西以及下周即将学的东西。自测后检查结果，确保准确知道哪些是自己知道的，哪些是自己不知道的。

用主要概念和术语背后的含义来考查自己，有助于把精力集中在核心思想，而不是次要的材料或是教室的措辞上。可以评估你学会了哪些内容，以及哪些内容是你还没有掌握的。练习回忆新知识可以让它们在记忆中更牢固，也有助于你在今后把它们回忆起来。也可以增强你把它们和先验知识联系起来的能力。

# 2.有间隔地安排检索练习

有间隔的练习意味着要不止一次地学习资 料，不过两次练习中间要隔开一段时间。

建立一份自测计划，在每个学习阶段之间都留出一段时间，具体多长时间取决于资料本身。

另外一种有间隔的检索练习方法是，穿插安排两个或更多的主题进行学习，这种轮换交替可以不断地刷新你对每个主题的记忆。

有间隔的练习让人感到操作起来更困难，因为此时有一点生疏，回忆知识的难度也增加了。正因为需要很大力气才能回想起来，这会强化记忆，强化对知识的掌握。

# 3.学习时穿插安排不同类型

轮换接触不同的问题和解法，把多种课题案例混合起来看。

不要每次只学习一种，可以轮换接触不同的问题和解法。

如果发现自己在固执地、反复地练习某个题目或某项技能，就要改变这种做法: 加入其他的科目、技能，持续不断地考验自己辨别问题类型的能力。

穿插练习可以把不同类型的问题或样本混合起来学习，提高你区别问题类型的能力，辩识出同一类问题的普遍特点，运用正确的解决方案。

# 4.细化知识连接已知

细化会提高你对新资料的掌握程度，能增加心智线索的数量，方便你在以后回忆并应用这些资料。

把资料和你已知的东西联系起来，用自己的语言向别人解释，或者解释这些资料与你的课外生活有什么联系。

反思把检索练习和细化结合起来，增添了学习层次，强化了技能。

老老实实地花时间真正写下答案。不写出答案，你就有可能陷入自以为知道的假象。

# 设计我的学习策略

在学习新知识的时候：

  1. 先理解主要含义，然后自己组织语言把概念描述清楚；

  2. 时常停下来，把知识拆分，细化，结合已知经历经验进行反思；

  3. 在学的过程中想几个问题测验自己，并把问题记录，下次用来自测；

  4. 给自己留有间隔时间去遗忘，尽量穿插着多样化的内容进行学习，并做周期性的检索练习；

  5. 查看自己的测验结果，准确的明白自己知道什么不知道什么。

针对不足的地方，重复上面这个过程。
