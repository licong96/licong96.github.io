# 算法工程师

主流应用：音频、视频、图像、数据挖掘、搜索

技术核心：机器学习

未来：人工智能

# 相关岗位

  1. 研究员
    • 博士起步，主要任务是看paper，研究前沿算法
  2. 算法工程师：
    • 一般硕士起步，将已有的算法模型以代码的形式实现落地，产生实际的效应
    • 比较关注工程，要对已有的算法有比较的了解
  3. 数据分析师
    • 一般硕士起步，寻找特征，分析数据，给算法工程师使用，更关注业务层
    • 对代码的要求不高
  4. 市场分析师
    • 商科出身，对业务有更深的了解，分析市场与实际调研
    • 对代码的要求进一步降低

# 互联网四大职位

  1. 开发工程师：有前端、后端，互联网公司最重要的岗位
  2. 数据工程师：有时与算法工程师混合，或者后端兼任
  3. 机器学习工程师：类似AI架构师，开发工程师加算法工程师，最顶尖的人才
  4. 产品经理：对产品进行设计，分析，项目。
