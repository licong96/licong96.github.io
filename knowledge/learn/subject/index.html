# 学科计划

问：你想要做什么？

答：我想要把人类已知的所有学科都了解一遍。

问：世界上的学科有一百多种，在有限的生命里，你怎么能做到？ 答： 世界上的学科的确非常多，可我的自身条件只有那么一点。我只会去尝试我能够摸得到的，在可选的范围内去逐一尝试。对于那些离我太遥远的，不太可能实现的，或者要非常困难才能实现的，我都不要。

问：为什么要这样做呢？ 答： 为了明月计划。为了认识自己。观察自己到底对什么学科有兴趣，对什么学科没兴趣。把这些都记录下来，用来帮我了解自己。

问：知道对哪些学科有兴趣之后呢？ 答： 我会记录下感兴趣的所有学科，从喜欢的里面筛选出更喜欢的，再从更喜欢的里面找出最喜欢的。这同时有助于我完成明月计划。

问：你的具体行动方案是怎样的？ 答： 我会把所有学科先整理出来，逐一了解，这里的了解只是看笼统的介绍，一边看一边就可以进行筛选，我这一步的筛选依据是我的先天条件。 然后我会得出一张写满学科的体验表，我会读这门学科中大概两本著作，在这个阶段，我先不会太深入的去研究，我只关心我在体验这门学科时，我的心里状态和全身感受是什么样的，都记录下来。

问：这样做有什么意义吗？ 答： 永远不要这样问，生命的本质是生物，现代生物学研究表明，生命只是盲目演化出来的一种结果，没有目的，没有终点，没有意义。反而是因为我去做了某件事，我才会觉得我段生命有点意义。

# 做法

  1. 先列出一个科目表，然后逐一尝试。
  2. 拿到一个科目后，先科普一番，它是什么？它能做什么？
  3. 记录自己在体验过程中的感受，喜恶，心得，行为。
  4. 结合自己的先天条件，出生，身体，资源。结合自己的后天优势，性格，特点，擅长。
  5. 体验一段时间后打一个综合分数。

# 时长

最短一周，最长一月。

这一步只是初步筛选，后面还会深入，所以不需要花太长时间在这一步，但是也不要在极短时间内就做出判断，稍微沉淀一下，防止误判。
