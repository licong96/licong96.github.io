# 购衣建议

买衣服，大多数情况下不是看到什么买什么，而是知道自己需要什么才买什么。

# 1.打造一个适合的衣柜

  • 随意一件单品都可以与三件以上的其他单品一起搭配。

  • 无论是在职场，社交，休闲还是居家的场景下，你都能在衣橱里找到合适的搭配。

  • 衣服整体的色调，设计，都能很好地反映出你自己的风格。

# 2.买衣服的加减乘除法

  • 穿衣做加法：一件衣服必须可以和三件衣服做搭配。

  • 买衣做减法：不能和衣橱中三件衣服搭配的就不要买。

  • 跨季节做乘法：同一件衣服可以穿多个季节那就是更棒的选择。

  • 购买价格不菲的衣服做除法：也就是说平均投资额＝原始价格／（使用次数＊年限）

# 3.衣橱固有单品的断舍离

  • 我仍然喜欢它吗？

  • 我有多久没有穿过这件单品了？

  • 对于我现在的体型、风格，这件衣服仍然合适吗？
