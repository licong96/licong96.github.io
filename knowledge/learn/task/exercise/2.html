# 健身

  1. 加入培训
  2. 获得国家证书
  3. 体验上岗

# 培训

选择被国家认证的培训机构，1-3个月，学费7000-16500。

# 证书

  • 国家资格认证
  • 四大认证
  • 急救证

培训拿国家资格认证可以上岗，自考拿国际ACE证书证明实力，拿急救证书保证安全。
