# 拓展

  • 编译型语言：C、C++

  • 解释型语言：JavaScript、Python

语言没有所谓的最好，适合场景才是最好。

运行效率与开发效率，鱼与熊掌不可兼得。通常开发效率高的语言，运行效率会比较低。

学习编程的经典误区：web 只是编程的一个应用方向，但不要把思维只局限在 web 上，世界上还有很多问题需要使用编程来解决。

当你遇到问题时，随手拿起 Python，编写一个工具来解决问题，这才是 Python 正确的打开方式。
